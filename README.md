# eM WhatsApp - Floating Button

A modern, professional, and fully customizable WhatsApp floating button plugin for WordPress with 2026 UI/UX design standards.

## Features

### 2026 Modern Design
- **Glassmorphism Effects** - Beautiful translucent design with backdrop filters
- **Smooth Animations** - Spring-based animations with optimized performance
- **Mobile-First Responsive** - Optimized for all devices with touch-friendly interactions
- **Dark Mode Support** - Automatic adaptation to user's color scheme preference
- **Accessibility Compliant** - Full keyboard navigation and screen reader support

### Advanced Functionality
- **Enable/Disable Toggle** - Control button visibility from admin panel
- **Custom Message Support** - Pre-fill WhatsApp messages for users
- **Position Control** - Choose between bottom left or right placement
- **Analytics Ready** - Built-in support for Google Analytics & GTM
- **Performance Optimized** - Lightweight with lazy loading and IntersectionObserver

### Admin Features
- Modern 2026 admin interface with intuitive controls
- Real-time toggle switch for enabling/disabling the button
- Custom phone number configuration
- Pre-filled message customization
- Position selection (left/right)

## Installation

1. Upload the plugin files to `/wp-content/plugins/simple-floating-whatsapp/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to **Settings > eM WhatsApp** to configure your button
4. Enter your WhatsApp phone number (with country code)
5. Toggle the button ON to display it on your website

## Configuration

### Phone Number Format
Enter your WhatsApp number in international format without spaces, dashes, or the + symbol:
- **Correct**: `35799123456` (Cyprus)
- **Correct**: `15551234567` (USA)
- **Incorrect**: `+357 99 123456`

### Custom Message
Add a pre-filled message that appears when users click the WhatsApp button. This helps initiate conversations:
- "Hi! I'm interested in your services..."
- "Hello! I'd like to get more information about..."

### Button Position
Choose where the button appears:
- **Bottom Right** (default) - Traditional placement
- **Bottom Left** - Alternative positioning
